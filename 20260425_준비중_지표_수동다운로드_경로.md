## 1. 기본 원칙

준비중 지표는 API보다 수동 다운로드 경로를 우선한다. 우선순위는 아래 순서로 둔다.

1. 대학알리미 공시데이터 다운로드
2. 대학재정알리미 통합공시/통계현황/대학재정데이터 첨부파일
3. 공공데이터포털 파일데이터 원문 다운로드
4. 위 경로에서 세부 컬럼이 없을 때만 API 또는 기관별 평가자료 요청 검토

## 2. 공통 다운로드 허브

### 2.1 대학알리미 공시데이터 다운로드

- URL: https://www.academyinfo.go.kr/popup/main0810/list.do
- 공공데이터포털 설명 페이지: https://www.data.go.kr/data/15081503/fileData.do
- 용도: 대학정보공시센터 공시데이터를 공시항목별 XLSX로 다운로드.
- 화면 절차:
  1. 대학알리미 접속
  2. 하단 또는 관련 메뉴의 `공시데이터다운로드` 진입
  3. `학교종류` 선택
  4. `항목명` 검색
  5. 공시연도/항목 선택 후 추가
  6. 하단 `Download` 또는 `다운로드` 클릭
- 주의: 화면 안내상 한 번에 최대 10개 공시항목까지 선택 가능하다.

### 2.2 대학알리미 대학주요정보 한눈에 보기

- URL: https://www.academyinfo.go.kr/main/main2130/doInit.do
- 용도: 주요지표 빠른 확인 및 엑셀 다운로드.
- 장점: 신입생 충원율, 재학생수, 장학금, 등록금, 전임교원 관련 대표 지표를 빠르게 받기 좋다.
- 한계: 현재 확보한 2026 후보 파일 기준 `재학생충원율`은 제공되지 않아 별도 공시항목 다운로드가 필요하다.

### 2.3 대학재정알리미

- 대학재정 주요 지표: https://uniarlimi.kasfo.or.kr/statistics/indicator
- 예·결산 결산 통합공시: https://uniarlimi.kasfo.or.kr/totalAnno/univFinance/accounts
- 교육시설 통합공시: https://uniarlimi.kasfo.or.kr/totalAnno/properties/facilities
- 대학재정데이터 자료실: https://uniarlimi.kasfo.or.kr/knowledge/data2Room
- 용도: 법인전입금, 장학금 지급률, 교비/법인회계 예·결산, 교육시설 현황, 교지·교사 첨부파일.

### 2.4 공공데이터포털 파일데이터

- 공공데이터포털에서 `공공데이터포털에서 다운로드(원문파일등록)`으로 표시되는 항목은 수동 CSV/XLSX 다운로드 경로로 사용한다.
- 기관 자체 URL로 연결되는 항목은 실제 파일은 대학알리미/대학재정알리미 화면에서 내려받는다.

## 4. 4주기 기준 3개 시설 지표 세부 경로

### 4.1 재학생 1인당 강의실 면적

- 기준 정의: 한국대학평가원 4주기 대학기관평가인증 평가 기준 양식 4.4-1.
- 필요한 컬럼:
  - 일반 강의실 면적
  - 멀티미디어 강의실 면적
  - 재학생 수
  - 캠퍼스 구분
- 우선 경로:
  1. 대학알리미 공시데이터 다운로드
  2. `학교종류=대학`
  3. 항목명 `교사시설` 또는 `교사(校舍)시설 확보 현황` 검색
  4. 최신연도 및 과거연도 선택
  5. 다운로드 후 강의실 세부 컬럼 확인
- 보조 경로:
  1. 대학재정알리미 → 통합공시 → 교육시설
  2. 연도 선택
  3. 조회항목 `교육용 건물(교사) 현황`
  4. 학교별/연도별 조회 후 다운로드 가능 여부 확인
  5. 대학재정데이터에서 `교사 현황` 검색 후 첨부파일 다운로드

### 4.2 재학생 1인당 실험실습실 면적

- 기준 정의: 한국대학평가원 4주기 대학기관평가인증 평가 기준 양식 4.4-2.
- 필요한 컬럼:
  - 계열: 인문사회, 자연과학, 공학, 의학, 예체능
  - 계열별 실험·실습실 수
  - 계열별 실험·실습실 면적
  - 계열별 재학생 수
  - 공동 사용 공간 배분 기준
- 우선 경로:
  1. 대학알리미 공시데이터 다운로드
  2. 항목명 `교사시설`, `실험·실습실`, `실험실습실` 순서로 검색
  3. 관련 항목을 연도별로 다운로드
  4. 계열별 면적 컬럼 존재 여부 확인
- 보조 경로:
  1. 대학재정알리미 → 대학재정데이터
  2. `교사 현황`, `교육용 건물 세부현황` 검색
  3. `최근 10개년 국·공·사립대학 교사 현황` 첨부파일 다운로드
- 리스크: 대학재정알리미의 교사 현황은 교육용 건물 세부현황일 가능성이 높아, 4주기 편람의 계열별 실험·실습실 면적까지 포함하는지는 파일 확인이 필요하다.

### 4.3 재학생 1인당 실험실습 기자재 구입비

- 기준 정의: 한국대학평가원 4주기 대학기관평가인증 평가 기준 양식 4.4-3.
- 필요한 컬럼:
  - 2024회계연도 계열별 실험·실습 기자재 구입비
  - 2025회계연도 계열별 실험·실습 기자재 구입비
  - 2024학년도 계열별 재학생 수
  - 2025학년도 계열별 재학생 수
  - 계열별 평균 등록금 또는 기준값 산출용 등록금
- 우선 경로:
  1. 대학재정알리미 → 통합공시 → 예·결산 → 결산
  2. 회계연도 `2024`, `2025`
  3. 회계구분 `교비회계`, 필요 시 `등록금회계`/`비등록금회계`
  4. 조회구분 `지출`
  5. `상세`에서 `기계·기구매입비` 또는 유사 계정 확인
  6. 학교별 조회 후 다운로드 가능 여부 확인
- 보조 경로:
  1. 대학알리미 공시데이터 다운로드
  2. `학생 1인당 교육비 산정근거`, `등록금 현황`, `재적 학생 현황` 다운로드
  3. 등록금 차액 기준값과 재학생 수 보정에 사용
- 리스크: 공개 회계자료의 `기계·기구매입비`는 전체 계정일 수 있으며, 4주기 편람이 요구하는 계열별 기자재 구입비는 대학 내부 작성자료 또는 평가 제출용 엑셀일 가능성이 있다.

## 5. 다음 확인 체크리스트

1. 대학알리미 공시데이터 다운로드에서 `교사시설`, `실험실습실`, `기계기구`, `학생 1인당 교육비 산정근거` 검색 결과 캡처/목록화
2. 각 항목 XLSX를 수동으로 다운로드해 컬럼 목록 확인
3. 계열별 컬럼이 없으면 대학재정알리미 첨부파일과 결산 상세자료로 보완 가능성 확인
4. 그래도 계열별 자료가 없으면 4주기 평가 제출 양식 기반 수동 입력 또는 기관별 자료 요청 경로로 분리
