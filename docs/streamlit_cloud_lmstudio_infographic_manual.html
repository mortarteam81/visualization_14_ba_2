AI 운영 방식 선택

현재 AI 기능은 LM Studio가 실행 중인 Mac과 외부 터널에 연결됩니다. Mac을 끄면 웹 앱은 접속되지만 AI 분석은 동작하지 않습니다.

A테스트 운영

Mac에서 LM Studio와 Cloudflare quick tunnel을 켜두는 방식입니다. 가장 빠르지만 터널 주소가 바뀔 수 있고 Mac이 켜져 있어야 합니다.

B상시 운영

항상 켜져 있는 서버나 별도 머신에 LM Studio 호환 API를 두고 고정 HTTPS 주소를 연결합니다. 운영 안정성이 가장 좋습니다.

C클라우드 LLM

OpenAI, Gemini 등 외부 LLM API로 전환하면 로컬 PC 의존성이 사라집니다. 별도 API 비용과 코드 설정 변경이 필요합니다.

운영자용

배포 후 필수 점검

앱이 보인다고 끝이 아닙니다. 로그인, DB, AI 분석까지 세 단계가 모두 통과해야 운영 준비가 끝납니다.

1로그인

Google 로그인 버튼이 보이고, 계정 선택 후 앱으로 돌아오는지 확인합니다.

  • 오류: redirect_uri_mismatch
  • 조치: Google Cloud Console에 Cloud callback URI 추가

2DB 연결

로그인 후 메뉴와 비교대학 설정이 정상 표시되는지 확인합니다.

  • 오류: Neon DB 연결 설정 필요
  • 조치: [connections.neon] url 확인

3AI 분석

AI 분석 실행 버튼을 눌러 LM Studio 응답이 생성되는지 확인합니다.

  • 오류: LM Studio 연결 상태 확인
  • 조치: 터널 URL, 모델 ID, 로컬 서버 확인
운영자용

Streamlit Cloud Secrets 구조

Secrets는 GitHub에 올리지 않습니다. Cloud 앱 Settings의 Secrets 화면에만 저장합니다.

[auth]Google 로그인 client_id, client_secret, redirect_uri
[connections.neon]Neon PostgreSQL 접속 URL
[app_auth]최초 관리자 이메일 목록
[lmstudio]LM Studio 외부 터널 URL과 모델 ID
client_secret, DB URL, API 토큰은 문서나 채팅에 그대로 붙여넣지 않습니다. 
[lmstudio]
base_url = "https://<터널주소>.trycloudflare.com/v1"
model = "supergemma4-e4b-abliterated-mlx"
# api_key = "인증을 켠 경우에만 입력"

Cloudflare quick tunnel 주소는 매번 바뀔 수 있습니다. 터널을 새로 열면 base_url도 다시 저장하고 앱을 재부팅합니다.

운영자용

LM Studio 임시 터널 재설정 절차

Mac을 껐다 켜거나 터널이 종료되면 아래 순서대로 다시 연결합니다.

1LM Studio 실행 후 Developer 탭에서 API Server를 켭니다.
2사용할 모델을 로드합니다. 현재 권장 모델은 supergemma4-e4b-abliterated-mlx입니다.
3로컬에서 http://127.0.0.1:1234/v1/models 응답을 확인합니다.
4cloudflared tunnel --url http://127.0.0.1:1234 명령으로 임시 HTTPS 주소를 만듭니다.
5Streamlit Cloud Secrets의 [lmstudio].base_url을 새 주소 + /v1로 바꿉니다.
6Secrets 저장 후 앱을 Reboot하고 AI 분석을 테스트합니다.
# 로컬 서버 확인
curl http://127.0.0.1:1234/v1/models

# 임시 터널 열기
cloudflared tunnel --url http://127.0.0.1:1234

# 터널 확인
curl https://<터널주소>.trycloudflare.com/v1/models
임시 터널은 테스트 편의용입니다. URL이 노출되면 외부에서 모델 API를 호출할 수 있으므로 장시간 운영에는 인증 또는 클라우드 LLM 전환을 권장합니다.

사용자용

화면별 사용 팁

대부분의 메뉴는 같은 패턴으로 작동합니다. 먼저 지표를 읽고, 그다음 비교와 AI 분석을 사용합니다.

A대시보드

핵심 지표를 한 화면에서 훑어보고, 특정 지표가 궁금하면 왼쪽 메뉴로 이동합니다.

B비교대학 설정

관심 대학과 비교 그룹을 저장하면 각 지표 화면의 차트와 AI 분석에 반영됩니다.

CAI 분석

분석 지표, 분석 톤, 분석 초점을 선택한 뒤 실행합니다. 결과는 요약, 기준 해석, 시사점, 권고 액션으로 표시됩니다.

문제 해결

오류 메시지별 빠른 판단

오류 문구를 그대로 운영자에게 전달하면 빠르게 조치할 수 있습니다.

접근 권한이 없습니다로그인한 Google 이메일이 허용 목록에 없거나 비활성화된 상태입니다.
redirect_uri_mismatchGoogle OAuth에 Streamlit Cloud callback URI가 등록되지 않았습니다.
Neon DB 연결 설정 필요Streamlit Cloud Secrets의 [connections.neon] url을 확인해야 합니다.
LM Studio 연결 상태 확인Mac의 LM Studio, Cloudflare 터널, [lmstudio] base_url, 모델 ID를 확인해야 합니다.
앱이 재시작 중입니다Secrets 저장, 코드 push, Reboot 직후에는 1-3분 정도 기다립니다.
사용자에게 권장되는 기본 조치: 새로고침, 로그아웃 후 재로그인, 오류 문구 캡처 후 운영자에게 전달.
