- Source: 대학알리미 공시데이터 추이
- Item: `5-다. 졸업생의 취업 현황_대학_졸업생 취업률`
- Raw file: `data/raw/academyinfo/jirosung/original/5-다. 졸업생의 취업 현황_대학_졸업생 취업률.xlsx`
- Candidate output: `data/conversion_outputs/academyinfo/jirosung/jirosung_2008_2024_candidate.csv`

## Candidate Columns

| Column | Meaning |
| --- | --- |
| `기준년도` | 졸업생 취업 현황 기준년도 |
| `학교명` | 학교명, 기본 분석 scope alias 기준으로 정규화 |
| `본분교명` | 본교/분교 구분 |
| `졸업생_진로_성과` | 취업자와 진학자를 함께 반영한 졸업생 진로 성과율 |

## Formula

- `취업률(%) = 취업자 / (졸업자 - 진학자 - 입대자 - 취업불가능자 - 외국인유학생 - 건강보험직장가입제외대상) * 100`
- `졸업생_진로_성과 = (취업자 + 진학자) / (졸업자 - 입대자 - 취업불가능자 - 외국인유학생 - 제외인정자) * 100`
- 공식 산식의 세부 취업 항목인 건강보험직장가입자, 해외취업자, 농림어업종사자, 개인창작활동종사자, 1인창(사)업자, 프리랜서는 대학알리미 학교 단위 XLSX에서 `취업자`로 집계 제공된다.
- 공식 산식의 `제외인정자`는 대학알리미 학교 단위 XLSX의 `건강보험직장가입제외대상` 컬럼으로 대응한다.
- 대학알리미 원자료의 `취업률(%)`은 원자료 산식 검증용으로 사용하고, 대시보드 지표는 취업과 진학을 함께 반영한 `졸업생_진로_성과`를 사용한다.

## Promotion Policy

이 schema는 후보 데이터 검증용이다. 운영 CSV인 `data/졸업생_취업률.csv`는 자동 교체하지 않는다.
