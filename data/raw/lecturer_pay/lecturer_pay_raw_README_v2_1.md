# 강사 강의료 원본 데이터

## 출처
- 사이트명: 대학알리미
- 운영기관: 한국대학교육협의회
- URL: https://www.academyinfo.go.kr/index.do#none
- 메뉴 경로: 공시데이터 다운로드 > 공시항목 > 14-차. 강사 강의료

## 수집 방식
- 2023년, 2024년, 2025년 3개년도 자료 다운로드
- 1개 파일로 통합
- CSV(UTF-8)로 변환

## 처리 개요
- 사용자 제공 통합 CSV 기준
- 한글 컬럼명을 전부 영문 snake_case 표준 컬럼으로 정리
- 데이터, 스키마, 출처, 처리 로직 파일 생성
