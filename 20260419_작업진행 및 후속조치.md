## 오늘 처리한 주요 오류 및 이슈

### 1. 선택 학교 강조 로직 관련 공통 오류

대표 증상:
- `TypeError: '<' not supported between instances of 'float' and 'NoneType'`
- `ValueError: dictionary update sequence element #0 has length 4; 2 is required`

원인:
- Plotly trace의 `opacity`가 `None`일 수 있었음
- `line`, `marker`, `margin` 등이 일반 dict가 아니라 Plotly graph object라 직접 `dict(...)` 변환 시 예외가 날 수 있었음

조치:
- `opacity=None` 처리 보완
- Plotly object를 안전하게 mapping으로 바꾸는 로직 추가
- 선택 학교 강조 스타일을 공통 함수 기준으로 정리

### 2. 논문실적 비교 대상 그룹 선택 수 제한 문제

원인:
- 논문실적 페이지만 그룹 멀티셀렉트 상태 동기화 방식이 달라 UI에서 선택 수가 비정상적으로 제한되는 문제가 있었음

조치:
- 그룹 선택 상태 관리 로직을 공통 유틸로 이전
- `*_group_schools_widget_*` 키 기반 동기화 구조로 통일

### 3. 법정부담금 CSV 인코딩 오류

대표 증상:
- `UnicodeDecodeError: 'cp949' codec can't decode ...`

원인:
- 실제 파일은 `utf-8-sig`인데 메타데이터는 `cp949`로 남아 있었음

조치:
- `registry/metadata.py`에서 `budam` 인코딩 수정
- `utils/data_pipeline.py` CSV 로더에 `utf-8-sig` 폴백 추가

### 4. 페이지 2~4 연도 컬럼명 불일치

대표 증상:
- `KeyError: '기준연도'`

원인:
- 실제 데이터셋 컬럼은 `기준년도`인데 리팩터링 과정에서 페이지 2~4 상수 일부가 `기준연도`로 바뀌어 있었음

조치:
- 페이지 2~4의 `YEAR_COL`을 모두 `기준년도`로 수정
