## 오늘 해결한 대표 오류

### 1. Plotly trace 스타일링 오류

증상 예시:

- `TypeError: '<' not supported between instances of 'float' and 'NoneType'`
- `ValueError: dictionary update sequence element #0 has length 4; 2 is required`

해결:

- `opacity=None` 방어 처리
- Plotly line/marker 객체를 바로 `dict(...)`로 취급하지 않도록 보완
- 선택 학교 강조 로직 공통화 시 trace 업데이트 방식을 안정화

### 2. CSV 인코딩 오류

증상 예시:

- `UnicodeDecodeError: 'cp949' codec can't decode ...`

해결:

- 법정부담금 CSV 인코딩 처리 보강
- `data_pipeline.py`에서 fallback 로직 반영

### 3. 컬럼명 불일치 오류

증상 예시:

- `KeyError: '기준연도'`

해결:

- 페이지별 실제 데이터셋 컬럼에 맞춰 `YEAR_COL` 해석 로직 점검
- 공통/페이지별 컬럼 선택 흐름 보완

### 4. AI 결과 스키마 변경 후 KeyError

증상 예시:

- `KeyError: 'management_implications'`

해결:

- 구버전 세션 결과와 호환되도록 `result.get(..., [])` 방식으로 렌더링 수정
