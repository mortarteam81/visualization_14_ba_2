## 오늘 처리한 주요 오류

### 1. 선택 강조 공통 오류

오류 예시:

- `TypeError: '<' not supported between instances of 'float' and 'NoneType'`
- `ValueError: dictionary update sequence element #0 has length 4; 2 is required`

원인:

- Plotly trace의 `opacity`가 `None`인 경우가 있음
- Plotly의 `line`, `marker`, `margin`이 일반 dict가 아니라 graph object라서 `dict(...)` 변환이 실패하는 경우가 있었음

조치:

- `opacity=None`이면 `1.0`으로 정규화
- `to_plotly_json()` 기반의 안전한 mapping 변환 함수 추가

### 2. 연구비 페이지 용어 불일치

원인:

- UI 문구가 `국내/국외`로 남아 있었음

조치:

- `교내/교외` 용어로 통일
- 비교 차트 제목, 탭 라벨, 기준선 라벨/색상까지 함께 수정

### 3. 등록금/기부금 페이지 AI 패널 연결 시 인코딩 문제

원인:

- 기존 파일에 깨진 컬럼명 문자열이 일부 남아 있어서 부분 패치가 불안정했음

조치:

- 두 파일을 짧은 구조로 다시 정리해 재작성
