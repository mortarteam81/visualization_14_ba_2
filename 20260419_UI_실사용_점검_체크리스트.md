# 2026-04-19 UI 실사용 점검 체크리스트

## 점검 원칙
- 순서: 페이지 1 → 7
- 각 페이지는 `진입`, `사이드바`, `메인 차트`, `보조 차트`, `AI 분석`, `오류 여부` 순으로 점검
- 코드 기반 자동 점검과 실제 브라우저 수동 확인 항목을 구분

## 공통 점검 항목
- 앱이 정상 기동되는가
- 페이지 진입 시 예외가 발생하지 않는가
- 사이드바 학교 선택이 정상 동작하는가
- 비교 대상 그룹 프리셋이 정상 반영되는가
- 메인 라인차트가 표시되는가
- 선택 학교 강조 색상/우측 라벨이 표시되는가
- 히트맵이 표시되는가
- 범프 차트가 표시되는가
- `선택 학교만 보기` 토글이 정상 동작하는가
- `Yearly stats`, `Year by school`, `Definitions` 또는 동등 섹션이 열리는가
- AI 분석 패널이 표시되고 입력/실행 흐름이 깨지지 않는가

## 페이지별 추가 확인

### 페이지 1. 법정부담금 부담율
- 비교 대상 그룹 평균선이 표시되는가
- 저구간 확대 보기 차트가 표시되는가
- 4주기 인증 기준선이 표시되는가

### 페이지 2. 전임교원 확보율
- 학생정원 기준 / 재학생 기준 전환이 정상 동작하는가
- 단일 학교 선택 시 기준별 비교 차트가 표시되는가

### 페이지 3. 연구비 수혜실적
- 교내 연구비 / 교외 연구비 탭이 정상 동작하는가
- 단일 학교 선택 시 교내/교외 비교 차트가 표시되는가
- 기준선 라벨이 `4주기 인증 기준(교내연구비)`, `4주기 인증 기준(교외연구비)`로 표시되는가

### 페이지 4. 논문실적
- 국내 등재지 논문 / SCI급·SCOPUS 논문 탭이 정상 동작하는가
- 단일 학교 선택 시 두 지표 비교 차트가 표시되는가
- 비교 대상 그룹 학교 수 제한 없이 선택 가능한가

### 페이지 5. 졸업생 진로 성과
- 분교 포함 토글이 정상 동작하는가
- 기준선 인근 확대 차트가 정상 동작하는가

### 페이지 6. 세입 중 등록금 비율
- 낮을수록 좋은 지표 특성이 히트맵/범프 차트에 올바르게 반영되는가
- 기준선 인근 확대 범위가 과도하게 넓거나 좁지 않은가

### 페이지 7. 세입 중 기부금 비율
- 높을수록 좋은 지표 특성이 히트맵/범프 차트에 올바르게 반영되는가
- 기준선 인근 확대 범위가 자연스러운가
