# 2026-04-21 작업 진행 및 후속 조치

## 작업 개요

오늘은 기존 시각화 시스템에 신규 지표 자료를 반영하고, 법정부담금 부담율 항목의 구조를 기준으로 교육비 환원율, 기숙사 수용률, 강사 강의료 항목을 시스템에 추가했다. 마지막으로 강사 강의료 차트의 표시 방식과 우측 라벨 동작을 정리하고 GitHub에 반영했다.

현재 로컬 `main` 브랜치는 GitHub `origin/main`과 동기화되어 있으며, 마지막 푸쉬 커밋은 `000774d Add lecturer pay metric`이다.

## 완료한 작업

- GitHub 저장소 `mortarteam81/visualization_14_ba_2`를 로컬 작업 디렉터리와 연결하고, 이후 작업분을 커밋 및 푸쉬했다.
- 교육비 환원율 자료를 `data/processed/education_cost_return_rate` 기준으로 시스템에 추가했다.
- 교육비 환원율 페이지와 레지스트리, 데이터 로딩 경로를 추가하고 기준값 `110%`를 반영했다.
- 기숙사 수용률 자료를 `data/processed/dormitory_accommodation_status` 기준으로 GitHub에 반영했다.
- 기숙사 수용률 페이지와 레지스트리, 데이터 로딩 경로를 추가하고 기준값 `11%`를 반영했다.
- 기숙사 수용률 페이지에서 `ValueError: 알 수 없는 dataset_key: dormitory_rate`가 발생한 원인을 확인하고, 데이터셋 키 등록 및 조회 경로가 정상 동작하도록 수정했다.
- 강사 강의료 자료를 신규 지표로 추가했다.
- 강사 강의료 산식은 `(시간당 지급기준 단가 × 총 강의시간 수)의 총합 / 총 강의시간 수의 총합`으로 반영했다.
- 강사 강의료 연도별 기준값은 `2023년 50,600원`, `2024년 51,800원`, `2025년 53,100원`으로 반영했다.
- 강사 강의료 처리 파이프라인에서 서울 소재, 사립, 대학교, 기존, 강사 기준 데이터를 필터링하고 학교별 가중 평균 강의료를 산출하도록 구현했다.
- 강사 강의료 페이지 `pages/10_강사_강의료.py`를 추가했다.
- 강사 강의료 차트에서 고정 수평 기준선 대신 연도별 기준값 trace를 표시하도록 조정했다.
- Plotly annotation 방식 라벨이 범례 숨김 상태와 어긋나는 문제를 피하기 위해 trace text 기반 라벨 방식으로 정리했다.
- trace text 라벨이 오른쪽 끝에서 잘리는 문제를 줄이기 위해 x축 오른쪽 패딩을 라벨 길이에 따라 동적으로 계산하도록 수정했다.
- 강사 강의료 차트에서 그룹 대학들이 기본적으로 보이고, 우측 라벨도 표시되도록 수정했다.
- 법정부담금 부담율 전용 차트 스타일러에도 동일한 trace text 및 x축 패딩 보정이 적용되도록 수정했다.
